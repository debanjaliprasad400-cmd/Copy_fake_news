# Fake News Detection using Machine Learning

This project detects whether a news article is Fake or True using NLP + ML models.

## Dataset
Dataset: Fake.csv + True.csv  
Source: [Google Drive Link](https://drive.google.com/file/d/1zam0nCZMt_udcDs1Le9p24ApYGuM2csX/view?usp=sharing)  
Note: Dataset hosted on Drive due to GitHub size limits.

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression, Decision Tree

## How to Run
1. Download dataset from Drive link above
2. Open `Copy of Q4-Fake_News_Detection...ipynb` in Google Colab
3. Mount Drive and run all cells
